App installers are intentionally not stored in GitHub Pages.

Upload release assets to GitHub Releases using these filenames:

- ChannelCrate-mac-apple-silicon-arm64.dmg
- ChannelCrate-mac-intel-x64.dmg
- ChannelCrate-windows-installer-x64.exe
- ChannelCrate-windows-portable-x64.exe

`release-config.js` is currently set to the `Release` tag for `lloganisawsome/ChannelCrate-`.
