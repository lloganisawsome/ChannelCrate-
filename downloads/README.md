App installers are intentionally not stored in GitHub Pages.

Upload release assets to GitHub Releases using these filenames:

- ChannelCrate-mac-apple-silicon.dmg
- ChannelCrate-mac-apple-silicon.zip
- ChannelCrate-mac-intel.dmg
- ChannelCrate-mac-intel.zip
- ChannelCrate-windows-setup.exe
- ChannelCrate-windows-portable.exe

Then edit `release-config.js` with the repository release URL.
